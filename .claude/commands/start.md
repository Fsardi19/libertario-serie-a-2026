# Protocolo de Apertura de Sesion

Ejecuta el siguiente protocolo de inicio:

## PASO 1: Cargar Contexto
Lee estos archivos en orden:
1. CLAUDE.md (raiz)
2. .claude/SESSION_LOG.md
3. .claude/PROGRESO.md

## PASO 2: Diagnostico Rapido
Ejecuta:
- ls -la
- ls -la .claude/
- git status (si aplica)

## PASO 3: Resumen para el Usuario
Presenta:
1. **Proyecto:** Nombre y proposito
2. **Ultima sesion:** Fecha, que se hizo, que quedo pendiente
3. **Estado actual:** % completado, proximos pasos
4. **Archivos clave:** Los mas relevantes para hoy

## PASO 4: Preguntar Objetivo
"Que quieres lograr en esta sesion?"

NO empezar a trabajar hasta recibir confirmacion del usuario.
